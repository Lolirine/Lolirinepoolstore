# 🌊 Lolirine Pool Store

Application web pour la gestion et la présentation des services et produits de piscine.  
Ce projet a été initialement développé avec **Google AI Studio** et est déployé sur **OVH**.

---

## 🚀 Fonctionnalités
- Présentation des services de pisciniste
- Catalogue des accessoires de piscine
- Formulaire de contact / demande de devis
- Interface responsive (mobile & desktop)

---

## 🛠️ Technologies utilisées
- **HTML5 / CSS3 / JavaScript** (Front-end)
- **Google AI Studio** (génération initiale du code)
- **OVH Hosting** (hébergement)
- **GitHub** (gestion de version)
